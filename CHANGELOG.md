# Changelog

All notable changes to this project will be documented in this file.

## [0.2.0] - 2026-05-18

### Added

- Create, update, and remove contacts
- Get a contact by contact ID
- Search contacts using keywords (first name, last name, email, or phone)
- Implement pagination for contact search results

## [0.1.0] - 2026-03-22

### Added

- Authentication using Passport JWT
- Implement refresh token for httpOnly cookie
- Get and Update basic information (email, username, first and last name) for current user
- English and Indonesian translation for authentication and users module

## [0.0.2] - 2026-03-09

### Changed

- Renamed package from `nest-contact-management` to `nestjs-contact-management` for naming consistency.

### Fixed

- Fixed `nestjs-i18n` loader path resolution for compiled development output.
